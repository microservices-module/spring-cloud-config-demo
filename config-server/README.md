# Config Server

This is a Spring Cloud Config Server microservice.

## Features
- Centralized configuration management for microservices
- Built with Spring Boot 3.x and Java 21

## Prerequisites
- Java 21
- Maven 3.8+

## How to Run
```
mvn spring-boot:run
```
